Evernote Statistics
===========================

ALPHA VERSION   


# Depends:  

* evernote-sdk-python
* cosm-python

# Installation on linux

git clone https://github.com/evernote/evernote-sdk-python.git  
cd evernote-sdk-python  
sudo python setup.py install  

git clone https://github.com/cosm/cosm-python.git  
cd cosm-python  
sudo python setup.py install
